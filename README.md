# google-adk-examples# 🤖 Google ADK Examples

A collection of practical examples and experiments using **Google Agent Development Kit (ADK)** for building AI Agents.

This repository demonstrates how to design, build, and experiment with AI agents using real-world patterns such as tool usage, memory, orchestration, and context management.

---

## 🎯 What This Repository Includes

**Input:** User queries, tasks, and prompts  
**Process:** LLM reasoning → Tool selection → Execution → Context handling  
**Output:** Agent responses, tool results, and automated workflows

---

## 📌 Examples Included

- Basic AI Agent using Google ADK 🧠  
- Tool-Using Agent (API / function calling) 🔧  
- Multi-step reasoning Agent 🔁  
- Context-aware Agent (Memory & Sessions) 🧩  
- Experimental Agent workflows 🚀
